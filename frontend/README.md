# Frontend (React + Vite)

This is the **frontend** for the MERN YouTube Clone.

## Run locally

```bash
npm install
npm run dev
```

The backend should be running on `http://localhost:5100`.

## Key pages/routes

- `/` Home page (video grid, search, filters)
- `/video/:id` Video player page (watch + comments)
- `/form` Login/Register
- `/channel/:id` Channel view

For full project documentation, see the top-level `README.md`.
